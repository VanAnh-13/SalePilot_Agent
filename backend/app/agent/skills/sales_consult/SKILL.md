# Sales consult

1. Hỏi số người, ngân sách, kiểu tủ và giới hạn kích thước nếu thiếu.
2. `delegate` → catalog với task rõ (dung tích/kiểu tủ + budget).
3. Gợi ý 1–3 SP + lý do ngắn.
4. CTA: kiểm tra giá/tồn thực tế / để SĐT / tạo đơn nháp.
